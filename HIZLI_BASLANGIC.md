- `index.html` içinde sertifika kartlarını kopyalayıp çoğaltın

## 🎨 Özellikler

### Dil Değiştirme
- Sağ üstteki **TR | EN** butonuna tıklayın

### Tema Değiştirme
- Sağ üstteki **🌙/☀️** butonuna tıklayın
- Koyu ↔ Aydınlık tema değişir

- Telefon ve tablette otomatik olarak uyum sağlar
- Mobil menü otomatik açılır

## ⚡ Hızlı Düzenleme İpuçları

### Metin Editörü Kullanın
- **Windows**: Notepad++ (ücretsiz)
- **Profesyonel**: VS Code (ücretsiz, önerilen)

### `<!-- TODO -->` Bulun
- Editörde **Ctrl+F** yapın
- `TODO` yazın
- Tüm doldurmanız gereken yerleri bulun

### Kaydet ve Yenile
- Değişiklik yaptıktan sonra **Ctrl+S** ile kaydedin
- Tarayıcıda **F5** ile yenileyin

## 🌐 Yayınlama (Deploy)

### Ücretsiz Seçenekler

#### 1. GitHub Pages (Önerilen)
1. GitHub hesabı açın
2. Yeni repository oluşturun
3. Dosyaları yükleyin
4. Settings → Pages → Enable

#### 2. Netlify
1. [netlify.com](https://netlify.com) hesabı açın
2. Portfolio klasörünü sürükle-bırak
3. Otomatik yayınlanır!

#### 3. Vercel
1. [vercel.com](https://vercel.com) hesabı açın
2. GitHub'dan import edin
3. Tek tıkla deploy!
